# 1. 概述

一般而言，在C++中如果在一对大括号中声明一个名字，则该名字的可见性就被限定在大括号的作用域范围内。但此规则却不适用于C++98风格的枚举型别中定义的枚举量。在C++98中定义的枚举，其枚举值/枚举量的作用域并非仅在定义该枚举量的大括号之内，其作用域范围与枚举定义的作用域范围相同：

``` C++

enum Color {red, green, blue};   // C++98风格的枚举定义
	// 此时的red green blue的作用域范围bu

```
